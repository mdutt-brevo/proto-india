**Heat glow strategy:**
The progressive glow is implemented as an animated `<radialGradient>` whose `cx`
attribute moves left-to-right over 1.5s. SVG gradient attributes are NOT animatable
via Motion directly. Use a `m.rect` overlay with `opacity` + `x` animation on a
color-stop shape instead:

- Place a `<linearGradient id="heatGradient">` in `<defs>` with stops:
  `transparent → COLOR.accentOrange@0.3 → COLOR.accentOrange@0.6 → transparent`
- Animate a `m.rect` filled with `url(#heatGradient)` using `x` (translateX) from
  `-100%` to `100%` over 1.5s with `ease: EASE_OUT_EXPO`. This gives a left-to-right
  sweep visually. Keep `x` in pixel units relative to the viewBox.

The barrel exterior pulse is a separate `m.rect` (barrel outline overlay) that
uses `opacity: [0.2, 0.7, 0.2]` with `repeat: Infinity` and `duration: 1.2` to
simulate the Tailwind `animate-pulse-glow` in Motion terms. Do NOT apply both CSS
keyframe animation AND Motion animate to the same element (anti-pattern: CSS + Motion
fight on compositor).

**Scene structure:**

```jsx
export default function MeltingScene() {
  // Container: stagger children into view
  const container = {
    hidden: {},
    visible: { transition: { staggerChildren: 0.08, delayChildren: 0.1 } },
  };

  // Barrel body fade-in
  const barrelVariant = {
    hidden: { opacity: 0 },
    visible: { opacity: 1, transition: { duration: 0.4, ease: "easeOut" } },
  };

  // Granule compress (scale down x-axis suggesting melting/merging)
  const granuleVariant = {
    hidden: { opacity: 0.9, scaleX: 1 },
    visible: {
      opacity: [0.9, 0.7, 0.3, 0],
      scaleX: [1, 0.8, 0.4, 0],
      transition: { duration: 1.2, ease: EASE_IN_OUT_SINE, delay: 0.3 },
    },
  };

  // Heat sweep: m.rect moves x from left to right
  const heatSweepVariant = {
    hidden: { x: -160, opacity: 0 },
    visible: {
      x: 160,
      opacity: [0, 0.8, 0.8, 0],
      transition: { duration: 1.5, ease: EASE_OUT_EXPO },
    },
  };

  // Barrel glow pulse (inline — cannot use Tailwind keyframe here without CSS class)
  // Use keyframes array pattern for ambient orange pulse
  const barrelGlowVariant = {
    hidden: { opacity: 0 },
    visible: {
      opacity: [0, 0.4, 0.2, 0.5, 0.2],
      transition: { duration: 1.5, ease: "easeInOut", times: [0, 0.2, 0.5, 0.8, 1] },
    },
  };

  return (
    <m.svg
      viewBox="0 0 400 300"
      xmlns="http://www.w3.org/2000/svg"
      aria-hidden="true"
      className="w-full h-full"
      variants={container}
      initial="hidden"
      animate="visible"
    >
      <defs>
        {/* Left-to-right heat sweep gradient */}
        <linearGradient id="heatGradient" x1="0%" y1="50%" x2="100%" y2="50%">
          <stop offset="0%"   stopColor={COLOR.baseBg}       stopOpacity="0" />
          <stop offset="30%"  stopColor={COLOR.accentOrange}  stopOpacity="0.3" />
          <stop offset="70%"  stopColor={COLOR.accentOrange}  stopOpacity="0.6" />
          <stop offset="100%" stopColor={COLOR.baseBg}        stopOpacity="0" />
        </linearGradient>
      </defs>

      {/* Barrel body — isometric horizontal cylinder shape */}
      {/* Main barrel rectangle */}
      <m.rect
        x="60" y="120" width="280" height="60"
        fill="none"
        stroke={COLOR.surfacePrimary}
        strokeWidth="2"
        rx="4"
        variants={barrelVariant}
      />
      {/* Barrel end cap left (feed throat) */}
      <m.rect
        x="50" y="115" width="20" height="70"
        fill="none"
        stroke={COLOR.surfacePrimary}
        strokeWidth="2"
        rx="3"
        variants={barrelVariant}
      />
      {/* Barrel end cap right (nozzle side) */}
      <m.rect
        x="330" y="125" width="30" height="50"
        fill="none"
        stroke={COLOR.surfacePrimary}
        strokeWidth="2"
        rx="3"
        variants={barrelVariant}
      />
      {/* Screw thread lines inside barrel (dashed, static) */}
      <m.line x1="80" y1="150" x2="320" y2="150"
        stroke={COLOR.textMuted}
        strokeWidth="1"
        strokeDasharray="6 4"
        variants={barrelVariant}
      />

      {/* Granule shapes inside barrel — compress and fade as they melt */}
      {[90, 120, 150, 180, 210].map((x, i) => (
        <m.rect
          key={x}
          x={x - 5} y="140"
          width="8" height="8"
          rx="2"
          fill={COLOR.surfacePrimary}
          opacity="0.8"
          variants={{
            hidden: { opacity: 0.8, scaleX: 1, scaleY: 1 },
            visible: {
              opacity: [0.8, 0.6, 0.2, 0],
              scaleX: [1, 0.7, 0.3, 0],
              scaleY: [1, 0.9, 0.6, 0],
              transition: {
                duration: 1.1,
                ease: EASE_IN_OUT_SINE,
                delay: 0.2 + i * 0.07,
              },
            },
          }}
        />
      ))}

      {/* Barrel exterior orange glow overlay (pulse) */}
      <m.rect
        x="60" y="118" width="280" height="64"
        fill={COLOR.accentOrange}
        rx="4"
        variants={barrelGlowVariant}
      />

      {/* Heat sweep — animated rect with gradient fill moves left→right */}
      <m.rect
        x="60" y="120" width="160" height="60"
        fill="url(#heatGradient)"
        rx="4"
        variants={heatSweepVariant}
      />
    </m.svg>
  );
}
```

**Critical rules:**
- Child `m.*` elements must use `variants` only — no `animate` prop on children.
- `initial` and `animate` are set ONLY on the root `m.svg`.
- Variants that need per-item delay use inline variant objects (not shared const) to
  allow per-item `delay` values inside the transition.
- Never animate SVG `fill` color directly — use `opacity` over a colored element.
- Never animate `width` or `height` — use `scaleX`/`scaleY` instead.
- Do NOT apply Tailwind `animate-pulse-glow` class AND Motion `variants` to the same
  element. Pick one system per element.

**No deferred features:** No annotation labels, no progress bar, no scene-aware palette shifts.
