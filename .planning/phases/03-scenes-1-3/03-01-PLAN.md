**Scene structure:**
- Root element: `m.svg` with `viewBox="0 0 400 300"`, `initial="hidden"`, `animate="visible"`, `xmlns="http://www.w3.org/2000/svg"`, `aria-hidden="true"`
- Container variant on `m.svg`: staggerChildren 0.04s, delayChildren 0.1s
- Particle variant: `hidden: { opacity: 0, y: -20, rotate: 0, scale: 0.6 }`, `visible: { opacity: 1, y: 0, rotate: randomDeg, scale: 1 }` with ease-in transition (duration 0.35–0.6s per particle)
- Hopper shape: static SVG `<polygon>` or `<path>` drawn as isometric industrial funnel — a trapezoid body with a narrow neck at the bottom, positioned in the lower half of the viewBox (approx y=160 to y=280). Use `stroke={COLOR.surfacePrimary}` and `fill="none"` for the technical-drawing look.
- Hopper interior rim: a thinner `<line>` or `<rect>` at the top opening to suggest depth.

**Particle generation:**
Generate an array of particle objects at module scope (outside the component) so they are stable across renders:
```js
function rand(min, max) { return Math.random() * (max - min) + min; }

const PARTICLES = Array.from({ length: 60 }, (_, i) => ({
  id: i,
  // x spread across hopper mouth opening (approx x: 120 to 280)
  x: rand(120, 280),
  // y start: above the hopper opening, scattered vertically
  yStart: rand(-40, 40),
  // y end: particle lands at hopper rim level (approx y=170)
  yEnd: rand(160, 180),
  // random rotation for tumbling effect
  rotate: rand(-45, 45),
  // stagger delay in seconds: 30–50ms spread
  delay: rand(0.03, 0.05) * i,
  // slight scale variation for organic feel
  scale: rand(0.7, 1.2),
  // width/height for rounded square: 4–8px
  size: rand(4, 8),
}));
```

**Component body:**
```jsx
export default function GranulesScene() {
  // Apply particle cap — slice stable PARTICLES array
  const isMobile = typeof navigator !== "undefined" && navigator.hardwareConcurrency < 4;
  const particles = PARTICLES.slice(0, isMobile ? 40 : 60);

  const container = {
    hidden: {},
    visible: { transition: { staggerChildren: 0.025, delayChildren: 0.05 } },
  };

  return (
    <m.svg
      viewBox="0 0 400 300"
      xmlns="http://www.w3.org/2000/svg"
      aria-hidden="true"
      className="w-full h-full"
      variants={container}
      initial="hidden"
      animate="visible"
    >
      {/* Isometric hopper: trapezoid body + neck */}
      {/* Wide top opening, narrow barrel feed at bottom */}
      <polygon
        points="120,160 280,160 250,240 150,240"
        fill="none"
        stroke={COLOR.surfacePrimary}
        strokeWidth="2"
      />
      {/* Hopper neck / barrel inlet */}
      <rect x="170" y="240" width="60" height="20" fill="none" stroke={COLOR.surfacePrimary} strokeWidth="2" />
      {/* Hopper depth rim (top rim back edge for isometric feel) */}
      <line x1="125" y1="156" x2="275" y2="156" stroke={COLOR.textMuted} strokeWidth="1" strokeDasharray="4 2" />

      {/* Granule particles — motion.div inside foreignObject for rounded squares */}
      {particles.map((p) => {
        const itemVariant = {
          hidden: { opacity: 0, y: p.yStart, rotate: 0, scale: 0.5 },
          visible: {
            opacity: [0, 1, 1, 0.6],
            y: p.yEnd,
            rotate: p.rotate,
            scale: p.scale,
            transition: {
              duration: 0.5,
              delay: p.delay,
              ease: "easeIn",
              opacity: { duration: 0.3, delay: p.delay },
            },
          },
        };
        return (
          <m.rect
            key={p.id}
            x={p.x - p.size / 2}
            y={p.yStart}
            width={p.size}
            height={p.size}
            rx={p.size * 0.25}
            fill={COLOR.surfacePrimary}
            opacity={0.9}
            variants={itemVariant}
          />
        );
      })}
    </m.svg>
  );
}
```

**Critical rules:**
- Child `m.rect` elements must NOT have their own `animate` prop — use `variants` only. Parent propagation drives the stagger.
- `initial` and `animate` are set ONLY on the root `m.svg`. All children use `variants` only.
- Use `m.rect` (SVG rect with rx for rounded corners) rather than `foreignObject` + `m.div` — SVG elements are simpler and avoid foreignObject cross-browser quirks.
- NEVER animate `fill` color in the variants — only `opacity`, `y`, `rotate`, `scale` (compositor-safe properties).
- Particles must use `COLOR.surfacePrimary` (slate #64748b) — no hardcoded hex.
