**SVG layout (viewBox="0 0 400 300"):**

The scene shows three elements in a horizontal arrangement:
1. Nozzle housing (left) — a short cylinder/rectangle in slate at ~x:40-100, y:125-175
2. Injection stroke path — orange `m.path` from nozzle tip (x:100, y:150) curving/straight to mould gate (x:280, y:150). A straight or slightly curved line conveys forceful linear injection.
3. Mould cavity (right) — an isometric rectangular block outline in slate at ~x:280-360, y:100-200, with a small gate opening at x:280, y:150

**Variant design:**

Use a container variant for stagger, but since the main element is a single path,
use `delayChildren` to let the background shapes (nozzle, mould) appear first, then
the stroke draws:

```js
const container = {
  hidden: {},
  visible: { transition: { staggerChildren: 0.1, delayChildren: 0.1 } },
};

const frameVariant = {
  hidden: { opacity: 0 },
  visible: { opacity: 1, transition: { duration: 0.35, ease: "easeOut" } },
};

// The injection stroke — pathLength 0→1 per user decision
const drawVariant = {
  hidden: { pathLength: 0, opacity: 0 },
  visible: {
    pathLength: 1,
    opacity: 1,
    transition: {
      pathLength: { duration: 1.2, ease: "easeOut", delay: 0.25 },
      opacity:    { duration: 0.15, delay: 0.2 },
    },
  },
};

// Trailing glow dot at nozzle tip (appears before stroke begins)
const nozzleGlowVariant = {
  hidden: { opacity: 0, scale: 0 },
  visible: {
    opacity: [0, 1, 0.6],
    scale: [0, 1.3, 1],
    transition: { duration: 0.4, ease: EASE_OUT_EXPO },
  },
};
```

**Critical rules:**
- `pathLength` is the ONLY way to animate the stroke draw. Do NOT use
  `stroke-dasharray` / `stroke-dashoffset` manually — Motion normalizes it.
- `initial` and `animate` are set ONLY on root `m.svg`. All children use `variants`.
- Stroke color is `stroke={COLOR.accentOrange}` set as a static SVG attribute.
  Do NOT animate `stroke` color through Motion — SVG stroke color animates on the
  main thread (repaint per frame). The static orange is correct.
- `strokeWidth` is a static SVG attribute (not animated). Use `5` for the forceful feel.
- Never animate `fill`, `stroke` (color), `width`, or `height` through Motion variants.
- Mould/nozzle shapes use `COLOR.surfacePrimary` (slate), not hardcoded hex.
